# Tic-Tac-Toe in Python

This project is an interactive Tic-Tac-Toe game built with Python.

The game allows two players to compete in a command-line environment, with automatic detection of wins, losses, and draws.

## Features
- Two-player gameplay
- Win and draw detection
- User input validation
- Clear and interactive game flow

## Purpose
This project was created to strengthen my Python programming skills and improve my understanding of game logic, loops, functions, and conditional statements.

## Technologies Used
- Python

## How to Run
1. Open the terminal  
2. Navigate to the project folder: cd Tic-Tac-Toe
3. Run the program : python main.py
